אלעזר

תהליך קליטת נתונים גולמיים:

המטרה לייצר רשימה של אובייקטים מקובץ Json

התרחיש:
המערכת מנסה לקרוא מקובץ ה- Json ולעשות Deserialization.
התהליך מסתיים ברשימה של אובייקטים קיימים,
.או בסיום מיידי של התוכנית עם הודעה ברורה של סיבת השגיאה

1. עבודה עם נתיבים
לקרוא קובץ ע"י נתיב שיעבוד בכל מערכת הפעלה,
ובכל מבנה קבצים

2. טיפול בשגיאות קריאה מקובץ
	a. בדיקה אם הקובץ קיים.
	b. בדיקה אם יש הרשאת קריאה.
	c. בדיקה אם ה - Json תקין תחבירית
	d. בדיקה אם Decerialization לא מחזיר null.
	e. בדיקה אם הקובץ מחזיר מערך ריק.
	

class DroneDataLoader

Method: Load(string filepath)
מקבל נתיב לקובץ
מנהל את כל תהליך הבדיקה
מחזיר: רשימה של אובייקטים,
או זורק שגיאה מתאימה.

Method: ValidatePath(filepath)
מקבל נתיב לקובץ
בודק אם הנתיב אינו ריק או null,
בודק אם הקובץ קיים.
זורק שגיאה אם True

Method: ValidateJsonContent(string contextfile)
מקבל את התוצאה של קריאת ה Json
בודק אם הקובץ ריק או null
זורק שגיאה אם True

Method: ValidateDecerialized(List<Drone>? drones)
מקבל את התוצאה של ה Deceralization
בודק אם התוצאה היא null
בודק אם הרשימה ריקה
זורק שגיאה אם True
